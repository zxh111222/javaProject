### 当一台电脑编辑完程序后，如何使用另一台电脑和git进行进一步的编辑？
1. 在Idea中打开终端  
2. 使用 `git pull origin master` 命令，其中 “master” 是本地仓库的分支名称
3. 每次更新完代码记得push到远程仓库
4. 当下次再换电脑写的时候，重复以上3个步骤
